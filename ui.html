prefers-reduced-motion

Cuando el sistema tiene activado "Reducir movimiento", el media query prefers-reduced-motion: reduce fuerza animation-duration: 0.01ms y transition-duration: 0.01ms en todos los elementos. Esto elimina el spring bounce, el spinner y el scale de botones sin modificar el código funcional.
